## 审查结论

- 已确认入口只做只读扫描:
  - 只遍历 `Application Support/AtlasX/logs`
  - 未新增任何日志写入、创建或轮转逻辑
- 已确认 CLI 与 HTTP 共用同一套日志状态读取主链:
  - 都调用 `logs.LoadStatus()`
- 已确认失败显式暴露:
  - logs root 不存在时显式返回 `present=false`

  - 非法 `limit` 显式返回错误
