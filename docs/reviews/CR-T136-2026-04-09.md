## 实现事实

- 发布证据脚本继续只复用 `e2e-gate.log` 作为未覆盖项的单一事实源
- `SUMMARY.md` 现在自动汇总：
  - `release_prerequisites`
- 当前规则映射：
  - `runtime verify smoke` -> 准备 staged managed runtime
  - `runtime install smoke` -> 准备 install plan 并设置 `ATLASX_E2E_ALLOW_INSTALL=1`
  - `tabs capture smoke` -> 启动受管浏览器会话并确保存在 page target
  - `browser-data open smoke` -> 保证受管会话可用且存在可打开的 browser-data
  - `sidebar ask real smoke` -> 配置真实 provider 并让 `sidebar_qa_ready=true`
- 当没有 `UNCOVERED` 项时：
  - 显式写 `release_prerequisites=none`
