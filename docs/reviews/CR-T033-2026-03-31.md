# CR-T033

- 任务: T033 用真实本机 Google Chrome.app 验证 managed stage 闭环
- 日期: 2026-03-31
- 结论: 通过

## 验证范围

- 真实源 bundle: `/Applications/Google Chrome.app`
- 验证环境: 临时 HOME `/tmp/atlasx-real-chrome-stage-home`
- 影响范围: 仅临时 HOME，验证结束后已删除

## 验证记录

1. `HOME=/tmp/atlasx-real-chrome-stage-home go run ./cmd/atlasctl runtime stage --bundle-path "/Applications/Google Chrome.app" --version "local-google-chrome"`
   - 退出码: 0
   - 摘要: 成功输出 staged bundle、binary、manifest 和 sha256。

2. `HOME=/tmp/atlasx-real-chrome-stage-home go run ./cmd/atlasctl doctor`
   - 退出码: 0
   - 摘要: 输出 `chrome_source=managed_auto`，`chrome_binary=.../Contents/MacOS/Google Chrome`，`managed_runtime_manifest_present=true`。

3. `HOME=/tmp/atlasx-real-chrome-stage-home go run ./cmd/atlasctl runtime status`
   - 退出码: 0
   - 摘要: 输出 `manifest_present=true`、`bundle_present=true`、`binary_present=true`、`binary_executable=true`。

4. `HOME=/tmp/atlasx-real-chrome-stage-home go run ./cmd/atlasctl launch-webapp --dry-run`
   - 退出码: 0
   - 摘要: 输出 `binary=.../Contents/MacOS/Google Chrome` 和 `runtime_source=managed_auto`。

5. `chmod -R u+w /tmp/atlasx-real-chrome-stage-home && rm -rf /tmp/atlasx-real-chrome-stage-home`
   - 退出码: 0
   - 摘要: 临时 HOME 已清理，未污染默认工作环境。

## 审查结论

- 当前开发机上真实安装的 `Google Chrome.app` 已可通过本地 stage 进入 managed runtime 链路。
- 初次清理临时 HOME 时，Go module cache 的只读权限导致 `rm -rf` 失败；补 `chmod -R u+w` 后已彻底清理。这是环境差异，不影响主工作区和产品逻辑。
