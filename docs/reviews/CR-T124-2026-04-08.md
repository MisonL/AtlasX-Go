## 实现事实

- `tabs` 主链新增：
  - `SetTitle(targetID, title)`
  - 先复用 `List()` 与 `findPageTarget(...)` 定位 page target
  - 再复用既有 page target websocket 通道执行 `Runtime.evaluate`
  - 仅显式设置 `document.title`，返回 `id/title/url`
  - `title` 为空白时显式失败 `title is required`
  - target 不存在或缺失 websocket debugger url 时显式失败
- CLI 新增：
  - `tabs set-title <target-id> <title>`
  - 输出 `id/title/url`
- HTTP 新增：
  - `POST /v1/tabs/set-title`
  - 请求体: `id`、`title`
  - 校验 `id/title` 必填，错误路径显式失败
