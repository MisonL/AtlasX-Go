## 实现事实

- `settings.Config` 新增：
  - `memory_persist_enabled`
  - 兼容旧配置缺省值为 `true`
  - `settings.View` 同步暴露该字段
- `memory` 主链新增：
  - `LoadControls(paths)`
  - `SetPersistEnabled(paths, enabled)`
  - `AppendPageCaptureControlled(...)`
  - `AppendQATurnControlled(...)`
  - `ParsePersistValue(...)`
- CLI 新增：
  - `atlasctl memory controls`
  - `atlasctl memory set-persist <enabled|disabled>`
- HTTP 新增：
  - `GET /v1/memory/controls`
  - `POST /v1/memory/controls`
  - 请求体要求显式提供 `persist_enabled`
- 写路径接入：
  - `tabs capture`
  - `sidebar ask`
  - `sidebar summarize`
  - `sidebar selection-ask`
  - 当开关关闭时动作仍执行，但不再写 `memory/events.jsonl`
