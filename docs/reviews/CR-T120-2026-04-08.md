## 实现事实

- `tabs` 主链新增：
  - `OpenDevToolsWindowIntoWindow(sourceWindowID, targetWindowID)`
  - 先校验源/目标窗口存在与差异性
  - 再对源窗口每个 page target 逐个复用 `OpenDevToolsInWindow`
  - 返回 `source_window_id/target_window_id/returned/opened_targets`
- CLI 新增：
  - `tabs open-devtools-window-into-window <source-window-id> <target-window-id>`
  - 输出窗口级批量结果与逐项 opened target
- HTTP 新增：
  - `POST /v1/tabs/open-devtools-window-into-window`
  - 请求体: `source_window_id`、`target_window_id`
  - 校验两个窗口 ID 都为正整数，错误路径显式失败
