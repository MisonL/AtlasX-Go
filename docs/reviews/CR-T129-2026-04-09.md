## 实现事实

- `tabs.Client` 新增：
  - `AuthMode(targetID)`
  - 复用既有 `Capture(targetID)` 主链抓取 DOM 文本
  - 通过第二次 `Runtime.evaluate` 只读读取 `document.cookie` 名称和 `localStorage/sessionStorage` 键
- 推断结果显式返回：
  - `mode`
  - `inferred`
  - `reason`
  - `host`
  - `path`
  - `login_prompt_present`
  - `workspace_signal_present`
  - `cookie_count`
  - `local_storage_count`
  - `session_storage_count`
- 推断规则：
  - 非 `chatgpt.com/openai.com` 域返回 `mode=unknown`
  - 登录页提示信号主导时返回 `mode=logged_out`
  - 工作区信号主导时返回 `mode=logged_in`
  - 信号混杂时返回 `mode=unknown`
- CLI 新增：
  - `atlasctl tabs auth-mode <target-id>`
- HTTP 新增：
  - `GET /v1/tabs/auth-mode?id=<target-id>`
- 结果明确标注 `inferred=true`
  - 不伪装成权威授权状态
  - 不提供登录/登出写路径
