# CR-T097

- 日期: 2026-04-08
- 任务: `T097 补齐批量按建议组整理到指定窗口入口`
- 结论: 已完成。`atlasctl tabs organize-into-window` 与 `POST /v1/tabs/organize-into-window` 已落地，CLI 与 HTTP 共用 `tabgroups.ApplyAllToWindow` 主链，先复用既有 `tabgroups.Suggest` 获取全部建议组，再按组顺序逐个复用 `applyGroupToWindow` 主链把全部建议组整理到指定现有窗口，并显式返回各组的 `moved_targets` 与 `aligned_targets`。

## 审查结论

- 已确认入口只复用现有两层主链:
  - 先走 `tabgroups.Suggest(...)` 获取全部建议组
  - 再按组顺序逐个走 `applyGroupToWindow(...)`
- 已确认 CLI 与 HTTP 共用同一套批量定向整理主链:
  - 都调用 `tabgroups.ApplyAllToWindow(...)`
- 已确认失败显式暴露:
  - `window_id` 非法时显式返回错误
  - `window_id` 不存在时显式返回错误
  - 任一建议组整理失败时显式返回错误
  - 已在目标窗口中的标签显式出现在各组 `aligned_targets`
