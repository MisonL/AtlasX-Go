## 实现事实

- `tabs` 主链新增：
  - `OpenDevToolsPanelInWindow(targetID, panel, windowID)`
  - 先复用 `DevTools(targetID)` 解析 frontend URL
  - 再复用 `resolveDevToolsPanelURL(...)` 编码 panel
  - 最后复用 `OpenInWindow(windowID, panelURL)` 在指定现有窗口内打开
- CLI 新增：
  - `tabs open-devtools-panel-in-window <target-id> <panel> <window-id>`
  - 返回 `window_id/activated_target_id/target`
- HTTP 新增：
  - `POST /v1/tabs/open-devtools-panel-in-window`
  - 请求体: `id`、`panel`、`window_id`
  - 校验 `window_id > 0`，错误路径显式失败
