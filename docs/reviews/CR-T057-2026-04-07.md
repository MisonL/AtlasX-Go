     - `runtime verify smoke`: 当前本机没有 staged managed runtime
     - `runtime install smoke`: 未设置 `ATLASX_E2E_ALLOW_INSTALL=1`
     - `tabs capture smoke`: 当前没有受管浏览器会话
     - `browser-data open smoke`: 当前没有受管浏览器会话

2. `cd atlasx && go test ./...`
   - 退出码: 0
   - 摘要: 全仓回归通过，gate 脚本和文档接入未破坏既有实现。

## 审查结论

- gate 现在有唯一入口，且明确区分了离线强制门禁与真实本机 smoke，不再把两者混成一套含糊口径。
- 缺失真实 runtime、受管浏览器或 provider 凭据时，脚本会输出 `UNCOVERED`，不会把未覆盖项伪装成通过。
- `runtime install` 仍是显式 opt-in，避免 gate 为了“跑全”而擅自修改当前机的 managed runtime 状态。
