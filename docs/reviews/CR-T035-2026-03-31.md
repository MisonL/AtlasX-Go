# CR-T035

- 任务: T035 为 atlasctl 增加 runtime verify 子命令
- 日期: 2026-03-31
- 结论: 通过

## 变更范围

- 新增 `managedruntime.Verify`
- 新增 `atlasctl runtime verify`
- 新增 manifest 缺失、binary 路径漂移、sha256 不一致的离线测试

## 验证记录

1. `cd atlasx && go test ./internal/managedruntime ./cmd/atlasctl`
   - 退出码: 0
   - 摘要: managedruntime verify 测试通过，覆盖成功、manifest 缺失、binary 路径漂移、sha256 不一致。

2. `cd atlasx && go test ./...`
   - 退出码: 0
   - 摘要: 全量 Go 测试通过，无回归。

3. `HOME=$(mktemp -d /tmp/atlasx-t035-missing-XXXXXX) go run ./cmd/atlasctl runtime verify`
   - 退出码: 1
   - 摘要: 输出 `manifest_present=false`、`verified=false`，并以 `managed runtime manifest is not present` 显式失败。

4. `HOME=$(mktemp -d /tmp/atlasx-t035-ok-XXXXXX) go run ./cmd/atlasctl runtime stage --bundle-path "/Applications/Google Chrome.app" --version "local-google-chrome"`
   - 退出码: 0
   - 摘要: 在临时 HOME 下完成真实本机 `Google Chrome.app` 的 managed stage。

5. `HOME=<临时HOME> go run ./cmd/atlasctl runtime verify`
   - 退出码: 0
   - 摘要: 输出 `manifest_sha256` 与 `actual_sha256` 一致，`binary_executable=true`，`verified=true`。

6. `chmod -R u+w <临时HOME> && rm -rf <临时HOME>`
   - 退出码: 0
   - 摘要: 临时 HOME 已清理，未污染默认工作环境。

## 审查结论

- `runtime verify` 不会伪造成功；无 manifest、路径漂移和 hash 漂移都会显式失败。
- 校验报告已经包含 manifest、bundle、binary、actual sha256 和 verified 状态，可直接复用于后续 HTTP API。
