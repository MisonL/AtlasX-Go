## 变更范围

- 在 `tabs` 层新增 `CaptureSelection(...)`，通过 CDP `Runtime.evaluate` 读取当前 page target 的浏览器原生选区
- 新增 `GET /v1/tabs/selection?id=<target-id>`，返回 `selection_text`、存在性、长度、截断与错误元数据
- `POST /v1/sidebar/selection/ask` 在未显式传 `selection_text` 时自动读取当前 tab 选区，仍保留显式字段覆盖能力
- 无选区时显式返回错误，不引入静默降级或伪成功路径
- README 与阶段事实文档同步补齐原生选区捕获能力
