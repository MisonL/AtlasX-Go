## 审查结论

- runtime 状态查询保持 `GET` 只读语义。
- 本地 stage 动作使用 `POST`，并复用 CLI 侧相同的 bundle 校验与 manifest 写入逻辑。
- 错误 bundle path 会返回明确错误 JSON，不存在伪造安装成功路径。
