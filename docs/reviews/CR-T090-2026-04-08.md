## 审查结论

- 已确认入口只复用现有两条主链:
  - 先走 `ActivateWindow(...)` 聚焦目标窗口
  - 再走 `Open(url)` 打开新标签
- 已确认 CLI 与 HTTP 共用同一套 open in window 主链:
  - 都调用 `tabs.OpenInWindow(...)`
