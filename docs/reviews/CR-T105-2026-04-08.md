# CR-T105

- 日期: 2026-04-08
- 任务: `T105 补齐默认浏览器只读观测入口`
- 结论: 已完成。`atlasctl default-browser status` 与 `GET /v1/default-browser` 已落地，CLI 与 HTTP 共用 `defaultbrowser.ReadStatus()` 主链，基于 macOS LaunchServices 当前 handler 读取 `http/https` 的 bundle id 与角色信息，只做只读观测，不执行设置默认浏览器写操作。

## 变更事实

- 新增 `defaultbrowser.ReadStatus()`
- 新增 CLI:
  - `atlasctl default-browser status`
- 新增 HTTP:
  - `GET /v1/default-browser`
- 新增测试:
  - `atlasx/internal/defaultbrowser/status_test.go`
  - `atlasx/cmd/atlasctl/default_browser_test.go`
  - `atlasx/internal/daemon/default_browser_test.go`
  - `atlasx/internal/daemon/default_browser_contract_test.go`

## 审查结论

- 已确认入口只做只读探测:
  - 仅读取 LaunchServices 当前 `http/https` handler
  - 未新增任何设置默认浏览器写操作
- 已确认 CLI 与 HTTP 共用同一套默认浏览器状态读取主链:
  - 都调用 `defaultbrowser.ReadStatus()`
- 已确认失败显式暴露:
  - LaunchServices 导出失败时显式返回错误
  - plist 转 JSON 失败时显式返回错误
  - JSON 解码失败时显式返回错误
  - 缺少 `http` 或 `https` handler 时显式返回 `unknown` 和 `known=false`
