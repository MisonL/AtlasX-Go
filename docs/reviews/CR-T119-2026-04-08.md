## 实现事实

- `tabs` 主链新增：
  - `OpenDevToolsInWindow(targetID, windowID)`
  - 先复用 `DevTools(targetID)` 解析 frontend URL
  - 再复用 `OpenInWindow(windowID, devtoolsURL)` 在指定现有窗口内打开
- CLI 新增：
  - `tabs open-devtools-in-window <target-id> <window-id>`
  - 返回 `window_id/activated_target_id/target`
- HTTP 新增：
  - `POST /v1/tabs/open-devtools-in-window`
  - 请求体: `id`、`window_id`
  - 校验 `window_id > 0`，错误路径显式失败
