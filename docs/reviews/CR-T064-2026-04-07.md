## 变更范围

- `atlasd` 新增监听地址校验，默认只允许回环地址
- 新增显式危险开关 `--allow-remote-control`
- 非回环地址在未启用危险开关时显式失败，不再静默监听
- `--once` 模式保持不变
- README 与 runbook 同步补齐监听边界说明
